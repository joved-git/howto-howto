Here you will find a Docker version of Kite. These scripts will create, update and start your Kite.

# Installation
This script creates the image, the container and starts Kite.
$ ./update-kite.sh


# Update the container
Use the same script to update Kite and start Kite.
$ ./update-kite.sh


# Start the container
Once the container was created, you can restart it with this command:
$ ./start-kite.sh
